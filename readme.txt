Команды которые могут быть использованы в файл менеджере
cd.. – перейти в папку на уровень выше
	Пример:
	cd..
cd <каталог> – перейти во вложенный каталог, 
	Пример:
	cd Program Files
md – создать новую директорию в текущей директории
	Пример:
	md Новая папка
cp '<путь который копируем>' <путь в который копируем> - копирование файлов и папок
	Пример:	
	cp 'C:\directory\Новая папка' 'C:\new directory\Новая папка'
	Пирмечание: пробел и кавычки обязательны
rm <файл, директория которую удаляем> – удалить файл или директорию (рекурсивное удаление –r)
	Пример:
	rm C:\directory\Новая папка
	Пирмечание: Можно ввести имя папки (без пути), которая находится в текущем каталоге
> - следующая страница дерева
	Пример:
	>
< - Предыдущая страница дерева
	Пример:
	>
quit - Завершить работу и выйти, с сохранением текущего каталога
	Пример:
	quit 

Примечание: Работает на Windows
